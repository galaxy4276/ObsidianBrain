# 문제 정의
해당 사회 내 교육 문제는 다음과 같다.
**디슬렉시아(난독증)를 가진 초등학생들을 위한 개인화된 읽기 및 어휘 학습 자료 생성 시스템 부족**
## 현황
디슬렉시아 학생들은 일반 교과서나 읽기 자료를 사용할 때 큰 어려움을 겪는다.
이들은 종종 정상적인 지능과 창의력을 가졌음에도 불구하고 텍스트 처리에 어려움을 겪어 학업 성취도가 낮아질 수 있다.

## 기존 해결책(난독치료) 의 한계
[(23. 08.) 난독증에 대한 오해와 효과가 의심스러운 치료 방법](https://21erick.org/column/11097/)

기존 난독아동에게 난독치료를 제공하는 절차는 학교 내 교사가 개인적으로 담당해주거나 난독 치료센터에서 지원받는 방법이 있다.
이 부분에 대한 한계는 다음과 같다.
- 현재의 디슬렉시아용 자료는 대체로 표준화되어 있어 개인별 특성(난독 유형, 강점 영역, 관심사)을 반영하지 못함
- 교사들이 개별적으로 자료를 수정하는 데 많은 시간이 소요됨
- 학생의 발전에 따라 자료를 지속적으로 업데이트하기 어려움

## AI 기반 해결 가능성
- 생성형 AI는 개별 학생의 디슬렉시아 특성(예: 시각적 처리 vs 음운적 처리 어려움)에 맞춘 자료 생성 가능
- 학생의 관심사(예: 공룡, 우주, 스포츠)를 반영한 텍스트 콘텐츠 생성으로 흥미 유발
- 특수 글꼴, 텍스트 레이아웃, 색상 대비 등 읽기를 돕는 시각적 요소 자동 조정
- 읽기 자료와 함께 보완적인 오디오, 그림, 애니메이션 등 다중감각 학습 자료 생성
- 학생의 진행 상황에 따라 난이도가 점진적으로 조절되는 적응형 콘텐츠 제공

## 기대 효과
- 조기 개입을 통한 디슬렉시아 학생의 학업 성취도 향상
- 읽기에 대한 좌절감 감소 및 자신감 향상
- 다양한 지역과 환경의 학생들에게 양질의 특수교육 자료 접근성 개선
- 교사의 개별화 교육 부담 감소로 더 많은 학생 지원 가능

이 아이디어의 사용 예시를 하나 들어보죠.

# 사용 예시 ) 디슬렉시아가 있는 초등학교 3학년 민준이

## 민준이
- 9세 남학생, 디슬렉시아 진단
- **정상 지능**이며 과학(특히 공룡)에 큰 **관심**이 있음
- **읽기 어려움**: 글자가 뒤집혀 보이고 단어를 인식하는 데 시간이 오래 걸림
- 시각적 정보 처리는 강점이나 긴 텍스트 처리는 어려움
- 현재 읽기 수준은 또래보다 **1.5년 정도 뒤처짐**
## 문제 상황
민준이의 3학년 과학 수업에서는 '생태계와 환경'에 대해 배우고 있습니다. 교과서는 긴 텍스트로 구성되어 있어 민준이가 따라가기 어려움.
담임 선생님은 30명의 학생들을 가르치면서 민준이에게만 별도의 자료를 만들어줄 시간적 여유가 없음.

## AI 솔루션 사용 과정
### 1. 초기 설정 및 프로필 생성
- 담임 선생님은 앱에 민준이의 기본 정보(나이, 학년, 디슬렉시아 유형, 읽기 수준)와 관심사(공룡, 과학)를 입력
- 앱이 민준이에게 몇 가지 짧은 읽기 테스트를 제공하여 정확한 읽기 패턴과 어려움을 분석

### 2. 교과 내용 변환
- 선생님이 '생태계와 환경' 단원 PDF를 업로드하거나 단원 주제를 입력
- #### **AI가 실시간으로 해당 내용을 민준이에게 최적화된 형태로 변환**
    - 글자 크기 확대, 디슬렉시아 친화적 글꼴(OpenDyslexic) 적용
    - 긴 문단을 짧은 단위로 분할하고 핵심 개념 강조
    - 주요 내용을 시각적 도표와 인포그래픽으로 변환
    - 민준이가 좋아하는 공룡을 예시로 활용한 생태계 설명 추가
    - 텍스트 내용을 음성으로도 들을 수 있는 오디오 옵션 제공

### 3. 개인화된 학습 활동
- #### **AI가 핵심 개념을 강화하는 인터랙티브 활동 자동 생성**
    - 공룡 시대의 생태계 변화를 탐구하는 짧은 만화형 이야기
    - 이미지 중심의 퀴즈로 개념 이해도 확인
    - 디슬렉시아 특성을 고려한 단어 연결 게임
    - 학습한 환경 개념을 공룡 세계에 적용하는 창의적 쓰기 과제(음성-텍스트 변환 지원)

### 4. 적응형 진도 관리
- 민준이의 활동 결과와 읽기 패턴을 분석해 다음 학습 자료 자동 조정
- 어려움을 보이는 개념은 다른 방식으로 다시 설명하고, 이미 습득한 개념은 심화 자료 제공
- 주간 진도 보고서를 선생님과 부모님께 자동 전송하여 민준이의 발전 상황 공유

## 기대효과

#### 민준이에게
- 읽기에 대한 좌절감 감소, 학습 내용을 실제로 이해할 수 있어 자신감 상승
- 자신의 관심사와 연결된 학습으로 과학에 대한 흥미와 호기심 유지
- 또래와 동일한 교과 내용을 학습하여 소외감 없이 수업 참여 가능
- 자신의 속도와 방식으로 학습하며 점진적으로 읽기 능력 향상

#### 선생님에게
- 민준이를 위한 특별 자료 제작에 들어가는 시간(주당 3-4시간) 절약
- 민준이의 학습 진도와 어려움을 데이터로 정확히 파악 가능
- 다른 학생들을 가르치면서도 민준이에게 질 높은 개인화 교육 제공 가능

#### 학교/교육 시스템에게
- 특수교육 전문가가 부족한 학교에서도 효과적인 포용 교육 실현
- 디슬렉시아 학생들의 학업 중단 위험 감소 및 학업 성취도 향상
- 학생 개인별 데이터를 통한 디슬렉시아 교육 방법론 개선

# 기술적 접근
### PDF 처리 및 텍스트 추출
- PDF 파일에서 텍스트, 이미지, 구조를 추출하기 위해 PDF 파싱 라이브러리(PyPDF2, pdf.js) 활용
- OCR 기술(Tesseract, Azure OCR 등)을 통해 스캔된 PDF에서도 텍스트 추출 가능

### AI 역할 구성
#### 1. 콘텐츠 변환 기능
- **원본 텍스트 분석**: 교과 내용의 복잡성, 주요 개념, 구조 분석
- **텍스트 최적화**: 디슬렉시아 친화적 텍스트로 변환 (문장 단순화, 구조 재조정)
- **컨텍스트 적응**: 학생의 관심사(예: 공룡)를 콘텐츠에 통합
#### 2. 시각화 자료 생성 기능
- **개념 시각화**: 추상적 개념을 시각적 표현으로 변환
- **인포그래픽 생성**: 복잡한 정보를 이해하기 쉬운 다이어그램으로 변환
- **스토리 이미지 생성**: 학습 내용을 보완하는 맞춤형 이미지 생성
#### 3. 학습 활동 제작기
- **퀴즈 생성**: 학습 내용 기반의 맞춤형 평가 문항 생성
- **게임화 변환**: 교육 내용을 인터랙티브 게임 형식으로 변환
- **스토리텔링 엔진**: 교육 개념을 스토리 형태로 재구성
#### 4. 피드백 분석기
- **학습자 반응 분석**: 학생의 활동 결과와 패턴 분석
- **난이도 조정 알고리즘**: 학습자 성과에 기반한 콘텐츠 난이도 동적 조정
- **진도 추적**: 개념 습득 수준 평가 및 학습 경로 추천
#### 5. AI 통합 오케스트레이터
- **API 호출 최적화**: OpenAI API 호출 관리 및 최적화
- **모델 선택 로직**: 각 작업에 가장 적합한 AI 모델 자동 선택
- **프롬프트 최적화**: 효과적인 결과를 위한 프롬프트 템플릿 관리 및 개선


### 백엔드 역할 구성
1. **API 통합 레이어**: AI 서버와 통신
2. **콘텐츠 처리 엔진**: 원본 교과 내용을 분석하고 재구성
3. **사용자 프로필 관리**: 학생의 특성, 선호도, 학습 진행 상황 저장
4. **적응형 학습 알고리즘**: 학생의 반응에 따라 콘텐츠 난이도 조정

### 프론트엔드 역할 구성
1. **교사용 대시보드**: 수업 자료 업로드, 학생 진도 확인
2. **학생용 인터페이스**: 디슬렉시아 친화적 UI/UX 디자인
3. **멀티미디어 플레이어**: 텍스트, 이미지, 오디오 통합 제공
4. **인터랙티브 활동 모듈**: 게임, 퀴즈, 스토리텔링 활동 구현

### 대회 기간 내 목표
- **MVP(최소 기능 제품) 완성**:
    - 기본적인 텍스트 변환 및 오디오 생성
    - 간단한 시각 자료 생성
    - 기초적인 학습 활동 2-3가지

## 시각화 요소 배치 알고리즘
전문지식이 부족해 AI 도움받은 것이니 참고만 부탁해요.
### 1. 시각화 필요성 판단 기준
- **개념 추상성 지수**:
    - 텍스트 내용의 추상성 수준을 NLP 기법으로 분석(0-10 척도)
    - 추상성 지수 7 이상: 시각화 우선순위 높음
    - 예: "생태계 평형"(추상성 높음) vs "북극곰"(추상성 낮음)
- **인지 부하 포인트 식별**:
    - 복수의 개념이 연결되는 지점 자동 감지
    - 긴 설명이 필요한 프로세스나 순환적 개념 식별
    - 텍스트 밀도가 높은 구간(단락당 정보량) 측정
- **핵심 개념 중요도 평가**:
    - 교과 커리큘럼 내 개념의 위계적 중요도 분석
    - 후속 학습에 재등장하는 기초 개념 식별
    - 평가 문항과 연계된 핵심 개념 우선 시각화
### 2. 최적 시각화 유형 선택
- **개념 유형별 시각화 매핑**:
    - 순차적 프로세스 → 플로우차트/타임라인
    - 구성요소 관계 → 계층도/마인드맵
    - 비교/대조 개념 → 벤다이어그램/비교표
    - 수치 데이터 → 그래프/차트
    - 공간적 관계 → 다이어그램/지도
- **학습자 선호도 반영**:
    - 개인 학습 데이터에서 드러난 효과적 시각화 유형 파악
    - 학생이 관심을 보인 시각 요소 스타일 우선 적용
    - 과거 퀴즈 성과가 높았던 시각화 방식 선호
