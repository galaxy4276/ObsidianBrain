# 서론
교내에서 사용하는 공인 네트워크 접근 여부를 판별하여 사용자에게 접근 권한을 부여하는 대용량 파일 공유 서비스이다.
2주 좀 넘는 짧은 시간동안 개발하였고 개인적으로 많은 것을 시도했으며 출시 및 사용자 활동 데이터까지 확보하여 많은 것을 느끼게 해주었다.
